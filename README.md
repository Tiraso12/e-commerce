## Installation
GIT CLONE the code or download the zip file, run npm install and make sure the connection to your database is correct!

RUN 'npm install'

LOG IN INTO  THE DATABASE AND  'source db/schema.sql'

RUN 'npm run seed' to seed database

RUN 'start sever'
